=====
Usage
=====

To use phuzzy in a project:

.. code-block:: python

    import phuzzy
    se = phuzzy.Superellipse(alpha0=alpha0, alpha1=None, m=2, n=None, number_of_alpha_levels=17)

.. code-block:: python

    import phuzzy
    tn = phuzzy.TruncNorm(alpha0=[2, 3], alpha1=[], number_of_alpha_levels=15, name="t")
